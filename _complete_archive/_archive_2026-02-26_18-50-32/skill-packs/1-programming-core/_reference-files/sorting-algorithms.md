# Sorting Algorithms

Implementation and comparison of merge sort, heap sort, and binary search.

## Merge Sort
- Time: O(n log n)
- Space: O(n)
- Stable: Yes

## Heap Sort
- Time: O(n log n)
- Space: O(1)
- Stable: No

## Binary Search
- Time: O(log n)
- Space: O(1)

## When to Choose
- **Merge Sort**: Need stability or external sorting
- **Heap Sort**: Memory constrained
- **Binary Search**: Fast lookup on sorted data
