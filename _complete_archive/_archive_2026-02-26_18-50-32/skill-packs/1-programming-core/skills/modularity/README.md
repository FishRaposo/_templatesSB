# Modularity Skill

Design modular systems with clear boundaries, reusable components, clean interfaces, and effective dependency management.

## Quick Start

Invoke this skill when you need to:
- Break a monolithic codebase into modules
- Define clean interfaces between components
- Manage dependencies and avoid circular imports
- Apply module systems (ES Modules, Python packages, Go packages, Rust crates)
