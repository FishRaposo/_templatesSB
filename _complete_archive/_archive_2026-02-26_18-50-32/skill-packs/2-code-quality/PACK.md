# Code Quality

**Pack ID**: 2-code-quality  
**Category**: Programming Fundamentals  
**Skills Count**: 12  

## Overview

This pack provides capabilities for writing clean, maintainable, and robust code. Every skill focuses on practical techniques that immediately improve codebases — from naming conventions and refactoring patterns to error handling, logging, and legacy migration.

## Skills Included

1. **clean-code** - Write readable, maintainable code with clear naming, small functions, and consistent formatting
2. **code-refactoring** - Improve code structure without changing behavior using proven refactoring patterns
3. **code-deduplication** - Eliminate duplicate code through abstraction, extraction, and shared utilities
4. **error-handling** - Implement robust error management with typed errors, recovery strategies, and fail-safe patterns
5. **input-validation** - Validate and sanitize inputs securely at every system boundary
6. **logging-strategies** - Implement structured, leveled logging with correlation IDs and observability
7. **code-quality-review** - Measure and improve code through structured review checklists and automated analysis
8. **technical-debt** - Identify, prioritize, and systematically reduce technical debt
9. **code-metrics** - Measure code quality with cyclomatic complexity, coverage, coupling, and churn analysis
10. **simplify-complexity** - Reduce unnecessary complexity through extraction, flattening, and decomposition
11. **code-standards** - Establish and enforce coding standards with linters, formatters, and CI gates
12. **legacy-code-migration** - Migrate legacy code safely with strangler fig, characterization tests, and incremental rewrites

## When to Use This Pack

Invoke skills from this pack when you need to:
- Improve readability and maintainability of existing code
- Refactor without introducing regressions
- Set up or enforce team coding standards
- Add proper error handling and input validation
- Implement logging and observability
- Assess and reduce technical debt
- Migrate or modernize a legacy codebase
- Prepare code for review or audit
