## Purpose

These files document the internal workings of the Universal Template System:

- **Blueprint Compilation**: How blueprints are compiled into project structures
- **Blueprint Mapping**: How blueprints map to stacks, tiers, and tasks
- **Changelog Generation**: Automated changelog creation from commits
- **System Integration**: How components integrate together
- **Tiered Templates**: MVP, Core, and Enterprise tier differences
- **Validation**: Template and documentation validation protocols
