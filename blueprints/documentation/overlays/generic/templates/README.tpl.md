# Reusable Templates

> Meta-templates for creating documentation structure

This directory contains reusable templates for common documentation patterns:

| Template | Purpose |
|----------|---------|
| [SUBDIRECTORY-INDEX.md](SUBDIRECTORY-INDEX.md) | Index template for directories with 5+ files |

## Usage

### Subdirectory INDEX Template

Use `SUBDIRECTORY-INDEX.md` when:
- A directory contains 5+ files
- Navigation assistance is needed
- File organization documentation is required

## Template Placeholders

All templates use `{{PLACEHOLDER}}` syntax for dynamic content:
- `{{DIRECTORY_NAME}}` - Name of the directory
- `{{FILE_COUNT}}` - Number of files in directory
- `{{LAST_UPDATED_DATE}}` - Last update timestamp
