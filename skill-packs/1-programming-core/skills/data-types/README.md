# Data Types Skill

Choose appropriate data types, handle type conversions safely, validate input, and build custom value types.

## Quick Start

Invoke this skill when you need to:
- Understand type systems across languages (dynamic vs static, structural vs nominal)
- Convert between types safely and explicitly
- Validate external data at runtime (Zod, Pydantic, etc.)
- Build custom value objects for domain concepts

## Example Usage

```
User: How do I validate API input in Python vs TypeScript?

Agent: In TypeScript use Zod schemas for compile+runtime safety.
In Python use Pydantic models with type hints for runtime validation...
```
