# Functional Paradigm Skill

Write clean, predictable code using pure functions, immutability, higher-order functions, and function composition.

## Quick Start

Invoke this skill when you need to:
- Refactor imperative code to functional style
- Build data transformation pipelines (map/filter/reduce)
- Implement pure functions and immutable data patterns
- Use currying, partial application, and composition
